# É
- Uma plataforma digital centralizada de análise de risco socioambiental e climático.
- Uma ferramenta que consolida e padroniza dados de diversas fontes.
- Uma solução intuitiva, acessível e orientada para diferentes usuários, como empresas, investidores e órgãos reguladores.
- Uma plataforma que gera scores baseados em critérios objetivos e verificáveis.

# NÃO É
- Um sistema de análise financeira ou de crédito puro.
- Um banco de dados estático ou apenas um repositório de informações.
- Um sistema exclusivo para especialistas técnicos ou restrito a um único setor.
- Um sistema subjetivo que atribui notas sem embasamento em dados concretos.

# FAZ
- Coleta e integra dados públicos de fontes confiáveis sobre riscos sociais, ambientais e climáticos.
- Gera um score de risco com base em algoritmos e modelos de análise.
- Permite a visualização dos dados de forma intuitiva, com gráficos e relatórios personalizáveis.
- Oferece insights para mitigação de riscos e tomada de decisão estratégica.

# NÃO FAZ
- Não realiza auditorias ou investigações próprias sobre empresas ou regiões.
- Não fornece apenas um relatório estático sem atualização contínua dos dados.
- Não apresenta informações de maneira técnica ou inacessível para usuários leigos.
- Não faz previsões financeiras ou avaliação de crédito diretamente.
